# Тестовая страница (Тема Магов)

Это тестовая страница, созданная из **Markdown** с темой Магов.

## Подраздел

Markdown позволяет писать:

- Списки
- **Жирный текст** и *курсив*
- [Ссылки](index.html)

> Цитаты оформляются через `>`.

Просто набери текст в `.md` файле, запусти `node convert-md.js` — и получи готовый `.html`.
